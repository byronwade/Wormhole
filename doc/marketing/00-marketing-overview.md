**Creative Professionals** (Video Editors, Game Developers, VFX Artists)

**Why:**
- Highest pain intensity (cloud costs, sync delays)
- Fastest buying decision (indie/freelance)
- Strongest word-of-mouth (active communities)
- Clearest "aha moment" (mount works â†’ instant convert)

### Pricing Structure

| Tier | Price | Target |
|------|-------|--------|
| **Free** | $0/forever | Individuals, viral growth |
| **Pro** | $8/user/mo ($6 annual) | Freelancers, power users |
| **Team** | $15/user/mo ($12 annual) | Studios, small teams (3-25) |
| **Enterprise** | Custom ($20-30/user/mo) | Organizations (50+) |

**Why These Prices:**
- 40-60% below cloud competitors (Dropbox $15, Box $20, Frame.io $25)
- Aligned with successful freemium SaaS (Notion $10, Slack $7-15, 1Password $8)
- P2P = 95% lower infrastructure costs = sustainable at lower prices
- Value: Users save $500-2,000/year in cloud subscriptions + hours of upload time

## Getting Started Checklist

### Pre-Launch (Do First)

- [ ] Finalize brand identity (logo, colors)
- [ ] Set up social accounts (Twitter, YouTube, Discord)
- [ ] Create landing page with email capture
- [ ] Write 3 comparison pages for SEO
- [ ] Create demo video (2-3 minutes)
- [ ] Prepare ProductHunt listing

### Launch Week

- [ ] Post to ProductHunt (Tuesday/Wednesday)
- [ ] Post to HackerNews (Show HN)
- [ ] Post to Reddit (r/VideoEditing, r/gamedev, r/selfhosted)
- [ ] Share on Twitter with demo video
- [ ] Email beta testers for support
- [ ] Engage with all comments/feedback

### Post-Launch

- [ ] Collect testimonials from early users
- [ ] Create first 3 case studies
- [ ] Start YouTube content calendar
- [ ] Begin outreach to creators/studios
- [ ] Set up analytics and tracking

## Key Messages by Audience

### For Video Editors
> "Stop waiting 30 minutes for your 50GB render to upload. Mount your render farm's output folder and start editing while files are still rendering."

### For Game Developers
> "Mount your build server's output folder - test instantly on all platforms. Replace Perforce for asset-only shares (free!)."

### For Privacy-Conscious Users
> "Your files never touch a third-party server. End-to-end encrypted. Open source. You control your data."

### For Cost-Conscious Users
> "Stop paying rent on your own files. Save $100-1,000/year vs. cloud storage. Free forever."

## Success Signals

### Positive Indicators
- Organic Reddit/Twitter mentions
- "How do I do X with Wormhole?" questions
- Requests for enterprise features
- Studios reaching out for case studies
- Users sharing join codes (viral loop working)

### Warning Signs
- High churn after first use
- "Couldn't figure out how to..." feedback
- "Why not just use Syncthing?" (positioning unclear)
- Zero word-of-mouth (product not solving real pain)
