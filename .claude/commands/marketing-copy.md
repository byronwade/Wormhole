## Brand Voice

- **Technical but accessible** - Explain without dumbing down
- **Direct** - Short sentences, no corporate speak
- **Slightly irreverent** - Question the status quo
- **Confident** - "It just works" not "We're the best"

- Primary: #7C3AED (Wormhole Purple)
- Accent: #EC4899 (Nebula Pink)

## Key Messages

### Value Props
- Mount any folder from any computer
- No cloud, no accounts, no fees
- E2E encrypted, open source
- Free forever (core features)

### Pain Points We Solve
- 50GB upload = 30+ min → Wormhole: <10 sec
- $50+/month cloud storage → Wormhole: $0
- Complex setup → Wormhole: share a code

### Don't Say
- "Revolutionary" → Say "Simple" or "Fast"
- "Seamless" → Be specific: "One-click"
- "Best-in-class" → Cite specific metric
